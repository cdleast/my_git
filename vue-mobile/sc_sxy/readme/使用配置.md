### 前端UI库 vant
* npm i vant -S

### 使用 vw 实现移动端适配
* cnpm i postcss-px-to-viewport --save-dev
* 在 package.json 中引用
```
"postcss": {
    "plugins": {
        "autoprefixer": {},
        "postcss-px-to-viewport": {
        "unitToConvert": "px",
        "viewportWidth": 375,
        "unitPrecision": 3,
        "viewportUnit": "vw",
        "minPixelValue": 1
        }
    }
}
```
