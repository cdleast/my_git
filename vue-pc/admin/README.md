# admin - 使用配置

#### 01、Element-UI 主题框架

- npm i element-ui -S

### 02、axios 前后端交互请求

- npm i axios -S

### 03、解决 ie9 不支持 es6 语法的问题

- npm i babel-polyfill -S
  import "babel-polyfill"

### 04、支持 ES6 语法=> 箭头函数可能会报 promise 错误

- npm i es6-promise -S
  import promise from "es6-promise"
  promise.polyfill()
